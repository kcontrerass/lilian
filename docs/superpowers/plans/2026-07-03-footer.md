# Footer Mockup Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implementar la nueva feature `footer` mobile-first, fiel al mockup, usando `features/footer/constants.ts` como única fuente de contenido e integrándola al final de la home.

**Architecture:** Se implementa un `FooterSection` como Server Component dentro de `features/footer/components/`, con layout mobile-first que escala a un grid de tres zonas en desktop. El componente consume `FOOTER_IMAGES`, `FOOTER_TEXTS` y `FOOTER_SOCIAL` para evitar hardcode de contenido y mantener Clean Architecture por feature. `app/page.tsx` solo se actualiza para orquestar la nueva sección como último bloque.

**Tech Stack:** Next.js 16 App Router, React 19 Server Components, Tailwind CSS v4 con tokens `@theme`, `next/image` para assets en `public/`.

---

## Global Constraints

- Mantener `app/page.tsx` como orquestador (sin markup de sección dentro de la página).
- Usar `bg-lilian-orange-dark` (token existente de `app/globals.css`) como color base del fondo del footer.
- Usar `next/image` con `sizes` para `FOOTER_IMAGES.main` y `FOOTER_IMAGES.logo`.
- Layout mobile-first, sin overflow horizontal en 390px.
- No agregar librerías nuevas de íconos/UI.
- Verificación final del proyecto: `npm run lint && npm run build`.

**Files:**

**Interfaces:**

- Produces: componente `FooterSection` reusable por la home.

- [ ] **Step 1: Crear el componente base e imports**

Crear `features/footer/components/FooterSection.tsx` con este bloque inicial:

```tsx
import Image from "next/image";
import { FOOTER_IMAGES, FOOTER_SOCIAL, FOOTER_TEXTS } from "../constants";

export default function FooterSection() {
  const socialLinks = FOOTER_SOCIAL.filter((item) => item.href && item.href !== "#");

  return (
    <footer className="relative w-full bg-transparent px-3 pb-3 pt-0 sm:px-4 sm:pb-4">
      <section className="relative overflow-hidden rounded-[2rem] bg-lilian-orange-dark text-white">
        {/* content */}
      </section>
    </footer>
  );
}
```
